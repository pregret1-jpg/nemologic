PATCH v1.2.0

네모로직 패치노트

2026.06.09 • UPDATE RELEASE

어제 진행된 힌트 완료 기능의 아쉬웠던 부분을 대폭 수정하고, 오직 플레이어의 직관적인 논리 추론만으로 퍼즐을 풀 수 있도록 자동 힌트 시스템을 완성한 대규모 정비 업데이트입니다.

주요 변경 사항

🧠 1. 유저 입력 기준 자동 체크 단일화
기존의 "정답 기준 자동 체크"와 "수동 체크" 모드를 완전히 배제하고, 오직 "사용자가 칠해 놓은 칸들의 크기 및 순서"만을 기준으로 힌트 완료를 판정하는 방식으로 룰을 일원화했습니다. 정답 위치를 컴퓨터가 미리 스포일러하지 않으면서도, 완성된 형태는 즉각 인지하여 편리하게 X(마크)를 채워줍니다.
🔄 2. 정밀한 양방향 Greedy 매칭 탑재
유저가 퍼즐을 왼쪽(위쪽)부터 순서대로 풀거나, 오른쪽(아래쪽)부터 순서대로 풀어나가더라도 상관없이 유연하게 감지하는 양방향 매칭 방식을 도입했습니다. 이를 통해 사용자가 아래에서부터 `2`와 `1`을 칠했을 때 힌트에서도 아래의 `2`와 `1`이 정확하게 체크됩니다.
🧭 3. 물리적 위치(중심선) 기반 모호성 제어
힌트의 숫자가 대칭적이거나 중복될 때(예: `4 1 1 4`) 어떤 힌트를 켜야 할지 모호한 경우, 사용자가 실제로 칠한 칸들의 격자판 상 중심 좌표를 자동으로 계산합니다. 칠해진 블록이 보드의 왼쪽 영역에 있다면 왼쪽 힌트 `4`를, 오른쪽 영역에 있다면 오른쪽 힌트 `4`를 선택해 켜주는 영리한 공간 감지 알고리즘을 도입했습니다.
🧹 4. 최적화 및 유휴 코드 제거
수동 체크용 메모 관련 CSS와 클릭 이벤트 리스너들을 모두 걷어내고, 사용하지 않는 구형 매칭 판정 함수(`getCorrectBlocksInfo`, `getMatchedHintIndicesCorrectness`)들을 대대적으로 청소하여 파일 용량을 최적화하고 렌더링 성능을 강화했습니다.

비하인드 노트

이번 패치를 통해, 사용자가 칠하면서 정답 위치 힌트를 얻어내는 다소 '쉬운' 난이도를 극복하고, 네모로직 실물 책이나 전문 퍼즐러들이 즐기는 "철저한 논리적 손맛"을 완벽히 살릴 수 있게 되었습니다. 공간적 치우침까지 파악하여 동작하는 스마트한 오토 체크와 함께 한층 깊어진 추론의 재미를 즐겨보세요!
